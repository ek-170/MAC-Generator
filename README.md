# MAC-Generator
Generate MAC address

```
Usage: mac-generator [options]
Options:
  -f string
        File format: csv, or json (default "csv")
  -format string
        File format: csv, or json. Defalut is csv (default "csv")
  -h    Show help information
  -help
        Show help information
  -hyphen
        Use hyphen (-) as delimiter in MAC addresses
  -n int
        Number of MAC addresses to generate. Default is 10
  -number int
        Number of MAC addresses to generate. Default is 10 (default 10)
  -o string
        Output file path and name (required)
  -out string
        Output file path and name (required)
  -s    Surround each MAC address with single quotes
  -surround
        Surround each MAC address with single quotes
```
